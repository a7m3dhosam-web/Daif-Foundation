# ⚡  ضيف للتوريدات الكهربائية - Daif Foundation

الموقع الرسمي لمؤسسة **ضيف للتوريدات الكهربائية** بمدينة العاشر من رمضان. تم تطوير هذا الموقع لعرض المنتجات الكهربائية وتسهيل عملية التواصل مع العملاء.

## 🚀 عن المشروع
هذا الموقع هو مشروع واجهة أمامية (Front-End) احترافي، مصمم ليكون سريعاً ومتجاوباً مع جميع شاشات الأجهزة (موبايل، تابلت، كمبيوتر).

### 🛠️ التقنيات المستخدمة:
* **HTML5:** لبناء هيكل الموقع.
* **CSS3:** للتصميم المتقدم والتنسيق الاحترافي (Responsive Design).
* **Google Fonts:** لتحسين تجربة القراءة للمستخدم.
* **WhatsApp API:** لربط الموقع مباشرة بخدمة العملاء.

## 📱 مميزات الموقع:
- [x] تصميم عصري واحترافي.
- [x] سهولة الوصول لأرقام الإدارة والمبيعات.
- [x] متوافق مع جميع مقاسات الشاشات.
- [x] أزرار طلب مباشرة عبر الواتساب لكل قسم.
